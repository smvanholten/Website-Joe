Firm Name Cleaning

This project provides tools to clean and harmonize firm names, either for general firm-name databases or specifically for the Burning Glass / Lightcast dataset of online vacancies.

Resources

  • Firm-name cleaning code (Stata .do file) — applies with minor modifications to any firm-name dataset that needs cleaning or harmonization.
  • Burning Glass / Lightcast crosswalk (3.5 GB .dta file) — maps raw firm names from the Burning Glass / Lightcast data to cleaned and harmonized versions, combining the cleaning code with machine-learning deduplication via the Dedupe algorithm.

Documentation

The training procedure and the construction of the crosswalk are described in Appendix A1.1 of National Wage Setting.

Authors

Anna Stansbury, Gregor Schubert, Jonathon Hazell, Christina Patterson, Heather Sarsons, and Bledi Taska.
